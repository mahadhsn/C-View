Welcome to C-View! Here's what you can do:

 ---------------------------------------
| Options - Usage: ./c-view [OPTIONS]   |
|---------------------------------------|
| --help        Show this menu          |
| -gs           Grayscale an image      |
| -rs           Reflect an image        |
| -rot          Rotate an image         |
| -bl           Blur an image           |
| -ed           Detect edges in an image|
 ---------------------------------------

Examples:
  ./c-view -gs [image file location] [image file output]
